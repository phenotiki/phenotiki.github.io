VERSION 1.2: 04/11/2016

  • General changes:
    • More errors/exceptions are now managed
  • Pot Tray Analysis:
    • Segmentation masks can now be exported individually
    • The importation of a segmentation mask caused the overwritten of previously collected data. This bug has been solved and the user can be asked whether they want to keep or overwrite data.
    • Datasets can be merged
    • New Image Importation System, allowing now any image format recognised by MATLAB
    • Fixed parameters selections for nested variables (e.g. DoG)
  • Leaf Annotation:
    • Added the possibility to massively transform annotations (e.g. rotate and translate)
  • Leaf Counting
    • Counting algorithm improved with extra features
    • Advanced parameters are initially hidden to avoid to overwhelm users
    • Added Random Forest Regression
    • Added more advanced parameters (e.g. settings for SVR and RF)
    • Added sanity checks on the advanced parameters when Start Training is pressed
  • Data Extraction:
    • CSV can be exported separately (as before) or altogether in the same file
    • Saved plots now have units (e.g., cm) in the y-axis

VERSION 1.1: 31/07/2016

  • General changes:
    • On the source code version, you cannot run some modules (e.g. Leaf Counting) if all needed dependencies are not met.
    • Added waiting window dialogs for long computations
  • Pot Tray Analysis:
    • Dataset can be reloaded with CTRL-R (CMD-R for Mac)
    • It is possible to change the timestamp of images and the dataset is automatically sorted
    • Fixed bug of conversion from pixels to cm2
  • Leaf Annotation:
    • Annotation can be showed/hidden (either using the checkbox or CTRL/CMD-H)
  • Data Extraction:
    • x-axis are formatted with dates
    • y-axis are converted in cm2
