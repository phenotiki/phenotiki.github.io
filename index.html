Phenotiki

True phenotyping-in-a-box solution

The Phenotiki platform provides a universal turnkey and modular platform for image-based plant phenotyping, using open source software and open hardware. It is an affordable and easy-to-use system for the acquisition and analysis of digital images, aimed at estimating plant growth. With one day setup time, and about €100 of equipment, which is easily installed, plants can be imaged and analyzed automatically in the cloud. No computers to maintain, no storage and archiving worries. Just Science!

Overview

Our approach to image-based plant phenotyping is based on a distributed sensing and analysis framework. Image acquisition relies on an affordable and versatile imaging sensor (based on the Raspberry Pi platform), that can be easily deployed for experiments in laboratories (growth chambers), greenhouses, or the field. Using state-of-the-art data compression, the acquired images can be efficiently transmitted over the Internet to the iPlant Collaborative cyberinfrastructure, offering cloud-based storage for life sciences research. The image data can be explored and analyzed using our web-based application (running on the Bisque platform), which provides an easy-to-use graphical user interface to perform sophisticated computer vision tasks. Our analysis system allows to delineate plant objects in the images and measure growth-related plant features (e.g., area, rosette diameter, relative growth rate, etc.). Phenotyping results are presented to the user for interpretation and can be downloaded for further statistical analyses.

Our solution is affordable and easy to deploy. We will soon provide detailed instructions to:

1

Install and configure the imaging setup, relying on a web-based interface allowing to contrl the sensor remotely (even from a smartphone).

2

Analyze the acquired image data using our cloud-based applications available on the Bisque framework of the iPlant Collaborative project.
